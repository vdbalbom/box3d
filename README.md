# 3D Box

### Linux
#### Install Lua
```
$ sudo apt-get install lua5.3
```
#### Install Löve
```
$ sudo apt-get install love
```
#### Clone this repo
```
$ git clone https://github.com/vdbalbom/box3d.git
```
#### Run
```
$ love box3d
```

### MacOs
#### Install Lua
```
$ brew install lua
```
#### Install Löve
```
$ brew cask install love
```
#### Clone this repo
```
$ git clone https://github.com/vdbalbom/box3d.git
```
#### Run
```
$ open -n -a love "box3d"
```

### Controls
| <b>Action</b>   | <b>Control</b> |
| -------------   | -------------- |
|    Move Right   |        →       |
|    Move Left    |        ←       |
|  Move Backward  |        ↑       |
|   Move Forward  |        ↓       |
|     Move Up     |        a       |
|    Move Down    |        z       |
